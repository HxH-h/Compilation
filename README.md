# 项目说明
学习编译原理时根据理解写的小工具
## FA
#### 文件结构
```
FA
    DFA.py   // 接收nfa状态，转为dfa并化简
    Edge.py  // 定义 边 类
    main.py  // 主函数，读取正规式，和画图
    NFA.py   // 接收正规式，预处理并转为nfa
```

#### 注意
* 自动将正规式转为NFA，再将NFA转为DFA，最后化简DFA
* 在main函数中输入正规式并执行main函数即可运行
* 支持多个终止状态但不支持多个开始状态
* 基于networkx库绘制图，但当有多重边出现时会被覆盖，后续会对此问题进行修复
